Problem

  • Customer: Library
  • Problem: People keep taking our books, so we don't know what books we have!
    • They do normally return the books after some time.

Requirements

What does our software need to do?

  • People need to be able to check out books.
  • People need to be able to return books.
  • Keep an inventory of the books.
  • Need to allow multiple copies per book.

Code Example

JavaScript

                
function multiply(a, b) {
  return a * b;
}

for(var a = 1; a <= 3; a++) {
  for(var b = 1; b <= 3; b++) {
    var c = multiply(a, b);
    console.log(`${a} * ${b} = ${c}`);
  }
}
console.log("Done!");
